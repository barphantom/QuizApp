# Django + React (Vite)

Projekt z backendem Django i frontendem React (Vite).

## Struktura

- `backend/` – Django + Django REST Framework
- `frontend/` – React z Vite


## Instalacja
```bash
cd QuizApp
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
## Jak uruchomić

### Backend
```bash
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
